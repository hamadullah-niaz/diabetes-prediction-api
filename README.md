# Diabetes Prediction API

This project predicts whether a patient is diabetic or not using Machine Learning and FastAPI.

## Technologies Used

- Python
- FastAPI
- Scikit-learn
- Pandas
- NumPy
- Joblib

## Features

- Diabetes prediction using Logistic Regression
- REST API using FastAPI
- API testing using Swagger Docs

## Run Project

```bash
uvicorn app:app --reload
